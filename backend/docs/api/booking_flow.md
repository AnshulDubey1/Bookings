# [DRAFT] Booking Flow

1. User visit website searches for a source and destination station enters quota and searches for the trains in that route
2. User is given a number of trains with there arrival date at the source and departure date at the destination along with classes of the train
3. User can check availability of the seats for that paticular route by clicking on the paticular class
4. If user wants to book a seat in the train the user must be signed in 
5. User fills the details or can be autofilled for the default user 
6. User submits and a seat is reserved atomically 
7. A confirmation page is displayed with the status of the booked ticket
